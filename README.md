# kaggle-titanic-decision-tree-analysis
Анализ данных и прогноз выживания пассажиров Титаника (Kaggle).

## Описание
В этом проекте проводится анализ, обработка пропусков и категориальных признаков, а также строится модель прогнозирования выживания пассажиров на основе деревьев решений для задачи бинарной классификации на основе открытого датасета из соревнования Kaggle - https://www.kaggle.com/competitions/titanic/data.

**Данные (`train.csv`, `test.csv`) не включены в репозиторий**. Ссылка для их скачивания и инструкции находятся внутри ноутбука.
