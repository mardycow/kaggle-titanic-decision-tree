# kaggle-titanic-decision-tree-analysis
Анализ данных и прогноз выживания пассажиров Титаника (Kaggle).

## Описание
В этом проекте проводится анализ, обработка пропусков и категориальных признаков, а также строится модель прогнозирования выживания пассажиров на основе деревьев решений для задачи бинарной классификации.
